# Financial Risk & Performance Metrics

A Python project for analyzing financial risk and performance metrics using historical stock data.

## Dependencies

- `yfinance>=0.2.32` - Download historical market data from Yahoo Finance
- `pandas>=1.5.0` - Data manipulation and analysis
- `numpy>=1.24.0` - Numerical computing
- `matplotlib>=3.7.0` - Plotting and visualization
- `scipy>=1.10.0` - Scientific and statistical computing
- `empyrical-reloaded` - Financial risk and performance metrics library
- `jupyterlab` - Interactive development environment

## Installation

```bash
# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
# Launch JupyterLab
jupyter lab
```

Then open `main.ipynb` to explore financial analysis examples.
